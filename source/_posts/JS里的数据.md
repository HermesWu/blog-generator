#### 数据类型

七种数据类型：number string boolean symbol undefined null object

1. number
    
    - 整数和小数： ` 1 1.1 .1`
    - 科学计数法：`1.23e2`
    - 二进制： `0b11`
    - 八进制： `011` `0o11`(ES5)
    - 十六进制： `0x11`

2. string

    - 空字符串： ''
    - 多行字符串： 

        ```
        var s = '12345' + 
                    '6789' //无回车符号
        或
        var s = `123456
        67890  // 包含回车符
        `
        ```
